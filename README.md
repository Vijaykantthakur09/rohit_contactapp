# Contact App (COM5013 Coursework)

A simple Contact Management application developed in Python for the COM5013 Algorithms and Data Structures module.  
The application allows users to add, view, delete, and list contacts through a menu-driven interface.  
This project demonstrates the use of appropriate data structures, algorithmic evaluation, and software testing.
